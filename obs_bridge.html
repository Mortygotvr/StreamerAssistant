Trigger Actions

Use * as a wildcard in the trigger. Example: Twitch * sub. You can use multiple triggers separated by commas.
New: You can update a Text Source dynamically! Use any attributes from the JSON payload like {username}, {message} (or {1}, {2} for text matched by the wildcards). To access custom data fields, you can use {customData.key}.
